# Structure

le travail ayant été fait sur le front uniquement, ce projet est un git parapluie reprenant le git du back d'origine, et celui du front retravaillé, le mien, afin de permettre le test fonctionnel de l'ensemble.

Les parties front et back ont leur README respectifs pour l'installation et lancement.

## Le travail fourni est consultable dans la partie frontend

Les blocs d'origine sont commentés pour pouvoir switcher au besoin et pour regénérer les rapports d'erreurs et les bugs à constater dans le débugger.

Les modifications sont relativement bien commentées
